In Windows XP, how do I access Sakai 2.0 using WebDAV?

On this page:

  • Introduction
  • Using My Network Places
  • Using Internet Explorer

Introduction

WebDAV stands for "Web-based Distributed Authoring and Versioning". It is a set of extensions to the HTTP protocol which allows users to collaboratively edit and manage files on remote web servers.

Using My Network Places

To access files in Resources using WebDAV via My Network Places, follow these steps:

  1. On the desktop, double-click My Network Places.

  2. In the Network Tasks pane, click Add a network place.

  3. When the Add Network Place Wizard appears, click Next.

  4. Select Choose another network location and then click Next. In the "Internet or network address:" field, enter the appropriate text:

    • If you know the correct URL, you can connect to your personal resource space by entering it into the "Internet or network address" field. Contact your site administrator for the URL's format; it will likely include your username.

    • Similarly, you can connect to the Resources for a specific course, and upload and download files to and from Resources, if you know the correct URL and the unique site ID number. The ID number is the sequence of numbers appearing immediately at the end of your site's URL. For example, once you have clicked the tab for a particular site, the URL field in your web browser will contain something similar to this: https://oncoursecl.iu.edu/portal/site/123456789-123

      The site ID in this case would be 123456789-123. To save a site ID number, while you are in the site, click File, then either Save as... or Save page as... . Make sure the site ID number is in the "File name:" field, and then save it to a place where you can easily retrieve it.

You may now drag and drop folders between your computer and the Resources to which you just connected. However, you cannot delete items in the Resources by dragging them to the Recycle Bin on your desktop. To delete an item, right-click it and select Delete.

Internet Explorer

You can also use Internet Explorer to open your files in Oncourse CL as a Web folder via WebDAV. Follow these steps:

  1. In Internet Explorer, from the File menu, select Open... .

  2. In the window that appears, click the checkbox next to Open as Web folder. In the "Open:" field, enter the appropriate text:

    • If you know the correct URL, you can connect to your personal resource space by entering it into the "Internet or network address" field. Contact your site administrator for the URL's format; it will likely include your username.

    • Similarly, you can connect to the Resources for a specific course, and upload and download files to and from Resources, if you know the correct URL and the unique site ID number. The ID number is the sequence of numbers appearing immediately at the end of your site's URL. For example, once you have clicked the tab for a particular site, the URL field in your web browser will contain something similar to this: https://oncoursecl.iu.edu/portal/site/123456789-123

      The site ID in this case would be 123456789-123. To save a site ID number, while you are in the site, click File, then either Save as... or Save page as... . Make sure the site ID number is in the "File name:" field, and then save it to a place where you can easily retrieve it.


  3. The contents of the resources folder will appear in a window where you can open individual files, browse to other folders on your computer, and transfer files between folders.

  4. When you're finished, close the browser.
