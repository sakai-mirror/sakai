Using Sakai 2.0, how do I add questions to a question pool or subpool?

To add questions to a question pool or subpool, follow these steps:

  1. In the menubar of the relevant course or project site, click Tests & Quizzes.

  2. From the Assessments page, click Question Pools.

  3. From the main Question Pools screen, click the name of the question pool to which you want to add the questions. If you'd like to add questions to a subpool, first reveal the name of the subpool by clicking on the triangle next to the larger pool. Then, click the name of the subpool.

  4. On the bar labeled "# Questions", where "#" is the number of questions in the pool or subpool, click Add.

  5. From the drop-down menu beside "Select question type", select the question type. Click Save.

  6. The precise options on the next screen will depend upon the type of question you are adding. Typically, you will indicate a point value for the question, the question text, correct answer(s), and feedback for correct or incorrect answers.
  7. To add the question to the pool or subpool, click Save.

  8. You will be returned to the main screen for your question pool or subpool. Under the question you just added, you can click Preview to preview the question.

  9. Click Add if you wish to add another question, and repeat the steps above. When you are finished, click Update.


  10. You can also add a question to a question pool when you create a new question in an assessment. Follow these steps:
    1. Create a new assessment.
    2. Create a new question in the assessment.
    3. Before saving the new question, find the "Assign to Question Pool" field.
    4. Click the drop-down arrow to show the list of Question Pools, and select the Question Pool that you want to add this question to.


    NOTE: When you add a question to a question pool while creating an assessment, and then later modify the question in the assessment, the question in the question pool is also modified.
