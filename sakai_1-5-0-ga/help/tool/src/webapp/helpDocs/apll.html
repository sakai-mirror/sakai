What it does

For courses, the Assignments tool allows instructors to create, distribute, collect, and grade online assignments. Assignments are private; student submissions are not visible to other users of the site.

The Assignments tool offers multiple grading options, including letter grades, points, check marks, pass/fail, or ungraded.

Assignments can also be returned, with or without grades, for re-submission. Instructors can download all submissions to an assignment to their computer at once. When instructors release grades for an assignment, students can access instructor comments and their grades.

Key concepts

Student view

As an instructor, you can view the assignments you create in student view so that you see what a student would see. To do so:
  1. From the menubar, click Assignments.

  2. From the View menu, select Student View.

  3. Click an assignment to see it in student view.

  4. Click Done to return to the assignment list. To return to instructor view, from the View menu, select List of Assignments.

Things to consider

  • Instructors can return assignments and allow for re-submission. This feature can be used to evaluate drafts of final projects or papers, or to allow students to correct and re-submit an assignment.
