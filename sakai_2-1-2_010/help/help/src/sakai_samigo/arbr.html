Using Sakai 2.0, how do I use the formatting toolbar while creating a question?

From the question editing screen, you can use the formatting tool bar (i.e., WYSIWYG editor) to control the graphical appearance of your questions and answers.

This toolbar is similar to those found within common word processing applications, and allows you to:

  • Format text to be bold, italicized, or underlined.

  • Change the face, size, and color of the font.

  • Left, center, or right-justify paragraphs.

  • Indent or un-indent paragraphs.

  • Insert a horizontal line.

  • Insert hyperlinks and images.

Note: The formatting toolbar will not work in Safari, but does work in Internet Explorer, Netscape, and Mozilla browsers.

Options and icons

The following options are available from the formatting toolbar:

  • Font: Choose from Ariel, Courier New, Georgia, Tahoma, Times New Roman, Verdana (this is the default font), Impact, and WingDings

  • Font size: Choose from a range of 8 to 36 points

Other formatting options and their corresponding icons are listed below:


Boldface text 		Boldface text: Changes the selected text to boldface

Italic text 		Italic text: Changes the selected text to italic

Underline text 		Underline text: Underlines the selected text

Strikethrough text 		Strikethrough text: Places a horizontal line through the middle of the selected text

Subscript 		Subscript: Creates subscript text, which is set lower than the baseline

Superscript 		Superscript: Creates superscript text, which is set higher than the baseline

Undo 		Undo: Undoes the action

Redo 		Redo: Repeats the action

Font color 		Font color: Gives font a color

Background color 		Background color: Provides background color for text

Flush left 		Flush left: Aligns the selected text on the left margin

Center 		Centered text: Centers selected text at equal distance from the left and right margins

Flush right 		Flush right: Aligns the selected text on the right margin

Justified 		Justified: Aligns selected text on both the left and right margins

Increase indent 		Increase indent: Increases the indentation of selected text

Decrease indent 		Decrease indent: Decreases the indentation of selected text

Horizontal rule 		Horizontal rule: Draws a horizontal rule on the selected line

Insert hyperlink 		Insert hyperlink: Changes selected text into a hyperlink

Insert image 		Insert image: Opens the "Insert Image" screen, from which you can select an image file to insert or point to a online image.

Help 		Help: Opens the Help screen containing descriptions of the various functions available on the toolbar

About 		About: Opens the About HTML Area information screen
