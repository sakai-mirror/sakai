Using Sakai 2.0, how do I set announcement permissions?

By default, any member of a worksite can read announcements, and site maintainers can read, create, revise, or delete any announcements. If you are a site maintainer, you can change these permissions by following the steps below:

  1. Enter a site for which you have a maintain role, click Announcements, and then click Permissions... .

  2. Place checkmarks in the appropriate boxes to set permissions. You will have the following options for different roles:

    • new
    • read
    • revise.any
    • revise.own
    • delete.any
    • delete.own
    • read.drafts

  3. Once you have set permissions for announcements, click Save.

  4. If you wish to exit and revert back to your current settings, click Cancel.
