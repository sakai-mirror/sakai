Adding an item

If you have the correct permissions, you can add an item (a file on your computer, a URL, a text document, or an HTML document) to a Drop Box or to a folder within a Drop Box. To do so, click Add Item under the title of the Drop Box or existing folder. Then follow the appropriate steps:

File upload

  1. On the Add Item(s) page, from the "Change Item Type" drop-down list, select File Upload.

  2. Use the drop-down list to select the number of files to upload. If you choose to add multiple files, repeat the following steps for each.

    Note: You can upload only 100MB of data at a time, so you may need to upload large files one at a time.

  3. Under "File Upload", click Browse... to find and select the file you want to upload.

  4. Under "Properties", type a title and description for your file in the appropriate fields.

  5. From the "Copyright Status" drop-down list, choose one of the following:

    • Materials in public domain.
    • I hold copyright.
    • Material is subject to fair use exception.
    • I have obtained permission to use this material.
    • Copyright status is not yet determined.
    • Use copyright below.

  6. Type any pertinent copyright information in the "Copyright Information" field.

  7. Use the checkbox to select whether or not to display the copyright alert and require acknowledgment of the copyright when others access the file.

  8. Click Add.

URL

  1. On the Add Item(s) page, from the "Change Item Type" drop-down list, select URL (link to website).

  2. In the "URL" field, type the URL for the website to which you're linking.

  3. Under "Properties", type a title and description for your link in the appropriate fields.

  4. Click Add.

HTML document

  1. On the Add Item(s) page, from the "Change Item Type" drop-down list, select HTML Page.

  2. Under "HTML Page", in the "Title" field, type a title for the page you're adding.

  3. In the provided text field, type the HTML code for your page. You can use the the WYSIWYG editor to help format your page.

  4. Under "Properties", type a description of your page in the "Description" field.

  5. From the "Copyright Status" drop-down list, choose one of the following:

    • Materials in public domain.
    • I hold copyright.
    • Material is subject to fair use exception.
    • I have obtained permission to use this material.
    • Copyright status is not yet determined.
    • Use copyright below.

  6. Type any pertinent copyright information in the "Copyright Information" field.

  7. Use the checkbox to select whether or not to display the copyright alert and require acknowledgment of the copyright when others access the file.

  8. Click Add.

Text document

  1. On the Add Item(s) page, from the "Change Item Type" drop-down list, select Simple Text Page.

  2. In the "Title" field, give your document a title.

  3. In the provided text field, type the text you wish to display.

  4. Under "Properties", type a description of your text in the "Description" field.

  5. From the "Copyright Status" drop-down list, choose one of the following:

    • Materials in public domain.
    • I hold copyright.
    • Material is subject to fair use exception.
    • I have obtained permission to use this material.
    • Copyright status is not yet determined.
    • Use copyright below.

  6. Type any pertinent copyright information in the "Copyright Information" field.

  7. Use the checkbox to select whether or not to display the copyright alert and require acknowledgment of the copyright when others access the file.

  8. Click Add.

Editing an item

To edit an item in a Drop Box or Drop Box folder, click Edit underneath the item's title, then do the following (depending on the type of item):

  • Uploaded files: You can click Browse... to find and select a replacement file. You may also edit the "Description" field, select another copyright status, edit the "Copyright Information" field, and choose whether or not to display the copyright alert. When you're finished, click Update.

  • URL: You may edit the "Title" and "Description" fields, and enter a replacement URL. When you're finished, click Update.

  • HTML or text document: From the "Content:" drop-down list, choose Revise document contents below, and then type your revisions in the provided text box, or choose Replace with document from my computer to choose a replacement file (see "Uploaded files" above). You may also edit the "Description" field, select another copyright status, edit the "Copyright Information" field, and choose whether or not to display the copyright alert. When you're finished, click Update.

Deleting an item

To delete an item in a Drop Box or Drop Box folder, click Delete underneath the item's title. On the confirmation page, click Delete again.
