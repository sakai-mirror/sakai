In Sakai 2.0, how do I add, edit, or delete a Drop Box folder?

Adding a folder

Your site's Drop Box should contain a folder for each participant. If you have the necessary permissions, you can add folders to the Drop Box or to an existing existing folder within the Drop Box. To do so, follow these steps:

  1. Under title of the Drop Box or existing folder, click Add Folder.

  2. On the Add Folder page, use the drop-down menu to select the number of folders to add.

  3. In the "Title" field, type the name of your new folder, and in the "Description" field, type a short description of the folder. If you're creating multiple folders, repeat this step for each of them.

  4. Click Add.

Editing a folder

If you have the appropriate permissions, you can edit a Drop Box or folder within a Drop Box. Under the title of the Drop Box or existing folder, click Edit, then on the Edit Folder page, you can modify the folder's title and description. When you're finished, click Update.

Deleting a folder

If you have the appropriate permissions, you can delete a Drop Box or folder within a Drop Box. Under the title of the Drop Box or existing folder, click Delete, and then confirm the deletion by clicking Delete again.
