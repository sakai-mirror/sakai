To add a new activity to your Schedule, follow these steps:

  1. In the menubar, click Schedule, then click New... .

  2. Type a title for the activity.

    Note: A red dot means that that information is required.

  3. Type a description for the new activity.

  4. To choose the date and start time for the activity, click the arrows.

  5. Click the Set frequency... button to select the frequency. Then enter information about how often the activity will occur and when it will end.

    Note: If you do not choose a frequency, your activity will occur once.

  6. Click the arrows to select the duration of the activity.

  7. To choose which type of activity will occur, pull down the Item type menu and select an activity type. It is optional to add a location for your activity.

  8. To save your activity, click Post.
