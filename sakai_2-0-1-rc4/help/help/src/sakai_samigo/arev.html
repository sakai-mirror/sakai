Using Sakai 2.0, how do I control who has access to my question pools?

You can control who can use or access your question pools from the main screen of the Question Pool Manager. Click User Admin to add or remove users, and to control access levels.

Adding users

To add users to a question pool, follow these steps:

  1. From the "User Admin" screen, click the Add User button that corresponds to the question pool title.

  2. Type the user's name and username in the appropriate fields.

  3. To set the access level for the user, click the radio button that describes the access level you want to grant:

    • 1- Create/Remove/View/Copy/Edit
    • 2- View/Copy/Edit
    • 3- View/Copy
    • 4- View only

  4. Click Save to add the user, or Cancel to abort.

Changing access levels

To change a user's access level, follow these steps:

  1. From the "User Admin" screen, click the question pool title bar to reveal the names of those allowed to access the pool.

  2. Use the radio buttons to change the level of access for each user.

  3. Click the Save button to save the change.

Removing users

To remove a user, follow these steps:

  1. From the "User Admin" screen, click the question pool title bar to reveal the names of those allowed to access the pool.

  2. Click the Remove button corresponding to the user you want to remove.

  3. On the resulting screen, click Remove to confirm and save your changes, or click Cancel to abort.
