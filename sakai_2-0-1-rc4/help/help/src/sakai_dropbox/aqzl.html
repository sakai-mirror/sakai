In Sakai 2.0, how do I revise information about a Drop Box or an item in a Drop Box?"

Follow these steps to revise a Drop Box's information:

  1. In the menubar, click Drop Box.

  2. Click Edit underneath the Drop Box whose information you wish to change.

  3. Edit the "Title" and "Description" fields as you like, then click Post to save the changes.

To revise the information about an item in a Drop Box, follow these steps:

  1. In the menubar, click Drop Box.

  2. Click the name of the folder containing the item that you want to edit.

  3. Click Edit underneath the item you want to revise.

  4. Edit the fields as you like, then click Post to save the changes.
