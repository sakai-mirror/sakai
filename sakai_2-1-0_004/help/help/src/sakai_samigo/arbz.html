Using Sakai 2.0, how do I edit metadata for my assessment?

Once you publish an assessment, you cannot change the metadata. Before you publish an assessment, you can edit its metadata from the Settings page. To do so, follow these steps:

  1. In the menubar of the relevant course or project site, click Tests & Quizzes.

  2. From the Assessments page, under "Published Assessments", click Settings beneath the title of your assessment.

  3. Click the black triangle next to "Metadata".

  4. Add any keywords, objectives, and rubrics in the provided text fields. Additionally, you can select Questions to record metadata for the questions in your assessment.

  5. Click Save Settings.

If you follow the instructions above and the metadata field is not accessible, you will need to create a new template that includes access to the metadata field.
