Using Sakai 2.0, in My Workspace, how do I change what appears in My Workspace Information?

By default, your My Workspace Information window contains announcements from the system administrator. To change what appears in My Workspace Information, follow these steps:

  1. In the My Workspace Information window, click Options.

  2. Under "Customize Frame", you can give the window a new title, and change the window's display height.

  3. Under "Description display", you can select one of the following options:

    • Display MyWorkspace information: Displays the My Workspace Information window
    • Display this worksite's description (from setup): Uses the default content (set by the system administrator)
    • Display this URL: Displays an alternate website that you enter in the "URL" text box

  4. Click Update Options.
