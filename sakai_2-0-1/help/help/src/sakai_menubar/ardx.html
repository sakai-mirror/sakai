In Sakai 2.0, using the Site Info tool, how do I add, edit, or delete the tools on the menubar?

Using the Site Info tool, you can add, edit, or delete the tools on the menubar. To do so, follow the steps below:

  1. In your site, from the menubar, click Site Info.

  2. Click Edit Tools.

  3. Check the boxes next to the tools that you wish to add to your menubar; uncheck the box if you are deleting a tool.

  4. Click Continue.

  5. If you're adding or already using the following tools, the Customize Tools screen will provide a chance to make certain changes. If you're not adding or using one of the tools below, skip to the next step. For the following tools, these tasks are possible:

    • Email Archive: Create or modify your site email address
    • News: Add or modify news feed titles and URLs; titles will appear in your menubar
    • Web Content: Add or modify titles and URLs; titles will appear in your menubar

    When you're finished, click Continue.

  6. On the confirmation screen, you will see a list of your site's tools and any changes you've made. If the list is accurate, click Finish. To edit your changes, click Back, or if you wish to cancel, click Cancel.
