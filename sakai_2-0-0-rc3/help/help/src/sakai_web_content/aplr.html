What it is

The Web Content tool allows site owners to choose a website to display within their worksite's Home frame.

Key concepts

You may display information in the Web Content window in different formats, including a text description, a web page, or an HTML file from your Resources.

Web Content link

The "Web Content" link in the menubar is customizable so you can create a label that matches the website you've chosen to display.

Things to consider

Instructors who want to provide many links to websites for their students may choose to list all of them in the Resources section, but only the most important ones with the Web Content tool.

Departments, collaborations, or other groups may use the Web Content tool to provide a prominent link to their public worksite.
