Worksite Information is content that is displayed in a box on your worksite's homepage. To add content to or edit what appears in your Worksite Information box, follow these steps:

  1. Open a course site or project site.

  2. If your site's homepage is not already open, click the Home button in the menubar.

  3. In the Worksite Information box, click the Options... button.

  4. To set the height of the box, under Customizing frame... , enter a pixel value in the "Height:" field.

  5. Enter the box's title in the "Title:" field.

  6. You can display either the text description that you entered when you created the site or a webpage. Follow the appropriate instructions below:

    To display the text description that you entered when you created the site, follow these steps:

    1. Select Display this worksite's description (from setup).
    2. To modify the description, from the menubar, select Site Info, and then edit the text in the "Description:" field. Click Continue, then Finish.

    You can display either an external webpage or an HTML file that you've created for this site. Follow the appropriate steps below:

    • To display an external webpage, select Display this URL. Enter a URL in the "URL:" field.
    • To display an HTML file, follow these steps:

      1. Upload your file to your worksite's Resources section.
      2. After you have posted your file to your worksite, if you do not see a list of your resources, from the menubar, click Resources.
      3. To get your file's URL, right-click (for Mac OS or Mac OS X, Ctrl-click) the filename and select the copy command for your browser (i.e., for Internet Explorer, select Copy Shortcut; for Netscape or Mozilla Firefox, select Copy Link Location).
      4. To return to your Worksite Information box, in the menubar, click Home, and then click the Options... button.
      5. Select Display this URL, and then in the "URL:" field, paste (for Windows, Ctrl-v; for Mac OS or Mac OS X, Cmd-v) the URL of your HTML file.

  7. To save your information, click Save. To return to the homepage without saving, click Cancel.
