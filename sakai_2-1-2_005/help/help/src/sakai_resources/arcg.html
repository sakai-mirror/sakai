Using Sakai 2.0, how do I create/add, edit, or delete a resource item?

On this page:

  • Creating/adding a new resource item
  • Editing a resource item
  • Deleting a resource item

Adding a resource item

Note: Fields marked with an asterisk are required.

Adding a new folder:

To add a new folder in which to place resource items, click the New Folder link beneath the existing level (i.e., existing folder) where you want to place the new folder.

Select the number of folders you want to create. For each one, enter a title and an optional description, choose your access, then click Add.

Adding resource items:

To create a new resource item, in the menubar, click Resources, and then under the folder in which you want to create the item, click Add Item. In the window that opens, from the "Change Item Type" drop-down list, select your resource type:

  • File Upload: An existing document or other file from your computer

    1. Choose the number of resources you would like to upload (you can upload up to 10).
    2. For each resource, click the Browse button, select a file to upload, and click Open (or you can type the name of the file in the space provided).
    3. Type a title for the resource (if you don't add a title, the filename will be used for the title) and add an optional description.
    4. Choose your file's copyright status and add optional copyright information (you can also optionally display a copyright alert), then choose the optional access selection.
    5. Click Add to finish.

  • URL (link to website): Specify a URL to a website that will be the resource

    1. For each resource, type the URL in the box. (You may omit the http:// prefix.)
    2. Add a title for your URL link resource. If you don't add a title, the URL will be used as the title of the resource.
    3. You can add an optional description, and choose your resource's access.
    4. Click Add to finish.

  • HTML Page: Create an HTML document

    Note: This feature will not work with Internet Explorer on the Macintosh.

    1. Enter a title, the content of your page, and an optional description in the appropriate text areas.
    2. Choose your page's copyright status, add optional copyright information, and choose whether you want to display a copyright alert.
    3. Select this resource's access.
    4. Click Add to finish.

  • Simple Text Document: Create a plain text document

    1. Type the title, content, and description in the appropriate text areas.
    2. Select the copyright status, enter optional copyright information, and choose whether you want to display a copyright alert.
    3. Choose your resource's access.
    4. Click Add to finish.

Editing a resource item

The edit option allows you to edit a resource from within the Resources tool. If you want to edit an existing resource, you can either revise it or replace it, depending on the type of resource. You can revise plain text and HTML documents; for other document types, the edit option only allows you to revise the information about the resource (e.g., the description), or replace the document's contents (e.g., replace a picture or URL). You can only replace document content with documents you upload from your own computer, one at a time.

To edit a resource:

  1. Click Edit beneath the resource that you want to revise or replace.

  2. If it is a document type for which you can revise the content, you will see a drop-down list labeled "Content:" from which you should choose either Revise document contents below or Replace with content from my computer. If it's a document type for which you can only replace content, click the Browse... button beside "Replace with this file:".

  3. For either option, alter any information about the resource you'd like to change as well, including the optional properties. (To view the optional properties, click the arrow beside "Optional properties".)

Deleting a resource item

From beneath the resource you want to delete, click Delete. A confirmation screen will appear asking if you are sure you want to delete the item. Click Delete to confirm, or click Cancel.
