Using Sakai 2.0, how can I copy Drop Box items to another folder?

To copy item(s) from one Drop Box folder to another, follow these steps:

  1. Click the black triangle next to the Drop Box containing the item you want to move.

  2. You will see all the items in that Drop Box. Click Copy beneath the item you want to move. You will then see this confirmation message: "Copied - Choose paste location, then click paste".

  3. Navigate to the new folder for the item. Click Paste Copied Item beneath it.

  4. If you wish, return to the folder in which the item was originally stored and click Delete.
