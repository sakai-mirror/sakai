On Sakai Collab Sakai Collab is a Sakai based service run by the Sakai Project to support Sakai development and use. To participate, create an account and join the Sakai Development site. You will find in-depth Sakai documentation in Resources, and receive Announcements (with email notification) about important Sakai developments. By joining this site you are automatically added to the sakai-dev@collab.sakaiproject.org mailing list.

CVS SVN Access

Sakai is now using Subversion! You can browse the code for this release using this URL:
  • https://source.sakaiproject.org/svn/tags/sakai_2-0-0

The same URL can be used with an SVN client to check out the code:
  • svn co https://source.sakaiproject.org/svn/tags/sakai_2-0-0

The Sakai development trunk is located here:
  • https://source.sakaiproject.org/svn/trunk/sakai

Requirements

Sakai requires Tomcat 5.5.9 or later, and the Tomcat compatibility package for use with Java 1.4:
  • Tomcat

Sakai also requires Java 1.4.2 or later (Note: Java 1.5 has not been tested with this release):
  • Java
Building Sakai requires, in addition, the Maven project management tool:
  • Maven

We recommend the Eclipse Java IDE for developing Sakai code:
  • Eclipse
