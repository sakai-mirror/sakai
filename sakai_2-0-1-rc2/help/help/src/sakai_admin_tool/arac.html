Using Sakai 2.0, how can I reset a user's password with the Users admin tool?

Note: To use this tool, you must have administrator rights.

To reset a user's password, follow these steps:

  1. From your My Workspace, click the Users tab. This will display a list of users defined within the system.

  2. Under "User Id", click the username of the user whose password you want to reset.

  3. In the "Create New Password:" field, type a temporary new password for the user. Re-type the password in the "Verify New Password:" field, and then click Save.
